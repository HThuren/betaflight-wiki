# OMNIBUS F7

F7 + OSD

## Features

- OSD
- SPI Gyro ICM-20608-G (SPI1)
- SPI Gyro MPU6000 (SPI3)
- STM32 F745 MCU
  - F7 MCUs have an integrated inverter, like the STM32F3
- SBUS/PPM input
- 4PWM output, DSHOT enabled
- BMP280 BARO
- VBAT sensor (voltage divider)
- External current sensor and RSSI ADC ready
- SDCARD
